# Trillo Hotel Booking UI

A modern responsive hotel booking user interface built with React and CSS.

## 🚀 Live Demo

https://trilo-project-with-react.vercel.app/

## ✨ Features

- Responsive Design
- Modern UI
- CSS Animations
- Flexbox Layout
- SVG Icons
- Component Based Architecture

## 🛠 Technologies

- React
- Vite
- CSS3
- HTML5
