# Trillo Hotel Booking UI

A modern and fully responsive hotel booking user interface built with React, Vite, and CSS. This project focuses on clean component architecture, responsive layouts, and smooth user interactions.

## 🚀 Live Demo

https://trilo-project-with-react.vercel.app/

## ✨ Features

- Responsive Design
- Modern UI
- CSS Animations
- Flexbox Layout
- SVG Icons
- Component Based Architecture

## 🛠 Technologies

- React
- Vite
- CSS3
- HTML5
-  JavaScript (ES6+)
